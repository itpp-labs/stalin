title: ЗАКУПНЕВ Захар Александрович
description: "Род. в 1890, русский, из крестьян, унтер-офицер, обр.: 1922-1924 окончил\
  \ Военно-Морскую Академию, член ВКП(б) с 1918. 1917-1919 военком Або-Аландской укрепленной\
  \ позиции, военком артиллерийского управления Петроградского ВО, 1919 командир батареи\
  \ форта Краснофлотский, 1920 начальник артиллерии крепости Очаков, военком крейсера\
  \ \"Нахимов\" (Червона Украина), 1921 помощник военкома Военно-морских учебных заведений,\
  \ 1922 военком Военно-морского училища им.Фрунзе, 1925-1930 помощник командира учебного\
  \ корабля \"9 января\", помощник командира минного заградителя \"25 октября\", помощник\
  \ командира, командир эсминца \"Сталин\", 1930-1931 командир дивизиона эсминцев,\
  \ 1931-1933 командир бригады эсминцев БФ, 1933-1935 командующий Северной военной\
  \ флотилией, награжден орденом Красной Звезды, 1935 капитан 1 ранга, 1935-1937 командир\
  \ 2-го дивизиона канлодок Каспийской военной флотилии \n  Арестован 19.03.1937.\
  \ Приговор: ВК ВС СССР, 04.09.1937 – ВМН. Расстрелян 04.09.1937. \n  Реабилитирован\
  \ 22.03.1958"
