  \ уездного РВК. 1921-1924-ответ.секретарь Южного бюро ВЦСПС, зам.председателя Южного\
  \ бюро ВЦСПС, кандидат в члены ЦК КП (б) Украины. 1924-1926-ответ.секретарь Одесского\
  \ губкома партии, зав.организационно-распредели-тельным отделом- секретарь, член\
  \ Оргбюро ЦК КП (б) Украины.1926-1928-ответ.секретарь Киевского окружкома партии-кандидат\
  \ в члены Политбюро ЦК КП (б) Украины. 1928-член Коллегии Наркомата торговли СССР.\
  \ 1928-1929-председатель ЦСНХ КАССР. 1929-председатель Комитета по делам печати\
  \ при Наркомторге СССР.1929-1930-председатель оргбюро Всесоюзного синдиката «Союзмясо».\
  \ 1930-1933-председатель правления треста «Союзмясо»-член Коллегии Наркомснаба СССР.\
  \ 1933-1934-начальник политурпавления Наркомата снабжения СССР. 1934-1938-начальник\
  \ Политуправления Наркомата пищевой промышленности СССР. \n  Репрессирован. Арестован\
