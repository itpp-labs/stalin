description: "Род. в 1889, г. Рига, латыш, из крестьян, прапорщик, член ВКП(б) с 1917.\
  \ Проживал: Москва, ул. Щукинская, 26-18-42. 1918-1920 помощник начальника, начальник\
  \ разведотдела 15-й армии, начальник 1-го отделения 1-го отдела, информационного\
  \ отдела, мобилизационного, организационного отделов Регистрационного управления\
  \ Полевого Штаба РВСР, 1921 начальник Курсов разведки, информационного отделения\
  \ разведотдела штаба помглавкома по Сибири, 1921-1923 в распоряжении Разведуправления\
  \ РККА, 1923-1933 помощник начальника 4-го отделения, уполномоченный 7-го отделения\
  \ секретного отдела ОГПУ, консул в Китае, Литве, Японии, 1933-1936 помощник начальника\
  \ по политчасти Курсов усовершенствования по разведке, 1936-1937 в распоряжении\
  \ Разведуправления РККА, 1937 цензор Отдела военной цензуры при Уполномоченном СНК\
  \ СССР по охране военных тайн в печати, полковой комиссар \n  Арестован 02.12.1937.\
  \ Обв. в шпионаже. Приговор: ВК ВС СССР, 07.05.1938 – ВМН. Расстрелян 07.05.1938.\
  \ \n  Реабилитирован 18.07.1956"
