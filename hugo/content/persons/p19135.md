title: ВОРОБЬЕВ Сергей Ильич
description: "Род. в 1903, из семьи учителя, б/п. 1920-1923 делопроизводитель, политработник\
  \ агитпункта Московского ВО, курсант 1-х Московских кавалерийских курсов комсостава,\
  \ курсант и старшина эскадрона 13-х Чугуевских кавкурсов комсостава, адъютант помощника\
  \ командующего Бухарской группой войск, командир и военком комендантского кавэскадрона\
  \ при штаба Бухарской группы войск, адъютант начальника Высшей стрелково-тактической\
  \ школы “Выстрел”, 1923-1924 командир отдельного кавэскадрона 17-й стрелковой дивизии,\
  \ помощник командира кавалерийского полка 17-й стрелковой дивизии, врид командира\
  \ 12-го кавалерийского полка, командир и военком 1-го кавалерийского полка 1-й Отдельной\
  \ Туркестанской кавалерийской бригады, врид командира 32-й кавалерийской бригады\
  \ 11-й кавалерийской дивизии, 1924-1926 военный советник в Китае, 1926-1935 помощник\
  \ командира 38-го кавалерийского полка, начальник штаба 38-го кавалерийского полка,\
  \ 63-го кавалерийского полка, помощник начальника отделения штаба Белорусского ВО,\
  \ помощник начальника отдела штаба Белорусского ВО, командир 31-го кавалерийского\
  \ полка, награжден орденом Красного Знамени (1924), орденом Красной Звезды 2-й степени\
  \ Бухарской Народной Советской Республики (1923), полковник, 1935-1937 инструктор\
  \ кавалерийской дивизии МНРА, \n  Арестован 14.11.1937. Приговор: ВК ВС СССР, 25.12.1937\
  \ – ВМН. Расстрелян 1937. \n  Реабилитирован 11.05.1957"
