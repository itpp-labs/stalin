---
title: БУРОВ Алексей Николаевич
description: Род. в августе 1893 в с. Тетюши Буинского уезда Симбирской губ. в семье
  крестьянина-середняка. Русский. Чл. РКП(б) с ноября 1918. В 1903 окончил 3-х годичную
  сельскую шк. в Тетюшах, в 1916 годичную военно-морскую шк. гальванеров (электриков)
  в Кронштадте, в 1931 курсы м.-л. при ЦК ВКП(б) в Москве. С 1903 по 1910 крестьянин
  в своем хозяйстве в Тетюшах. В мае-августе 1910 землекоп на стр-ве Амурской ж.д.
  С августа 1910 по январь 1911 шахтер Черновских рудников Забайкальской обл. С января
  1911 по май 1912 пекарь булочной торговой конторы в Чите. В мае-декабре 1912 пекарь,
  кладовщик булочных Чебаненко, Солянова во Владивостоке. С декабря 1912 по апрель
  1914 крестьянин в своем хозяйстве в Тетюшах. С мая по октябрь 1914 пекарь, кладовщик
  нарядчика Солянова во Владивостоке. С ноября 1914 по май 1916 военный моряк, ученик
  шк. гальванеров в Кронштадте. С мая 1916 по март 1918 гальванер линкора "Гангут"
  Балтийского флота. С марта 1918 по декабрь 1921 работал в Н.Новгороде. В марте 1918
  - феврале 1920 зам. пред., чл. приемной комиссии военно-морского порта. С февраля
  по август 1920 секретарь, зам. нач., чл. для особых поручений обл. упр-я водного
  транспорта (Облполитвод) Волжского бассейна. С сентября 1920 по сентябрь 1921 чл.
  для особых поручений склада Упр-я чрезвычайного уполн. Совета рабочей и крестьянской
  обороны (СТО) по снабжению Красной Армии и Флота (Чусоснабфлот). В сентябре-декабре
  1921 комиссар радиомастерских. С декабря 1921 по май 1922 уполн. Упр-я губкомиссии
  СТО в Симбирске. С мая по июль 1922 уполн. комиссии СТО по учету имущества в Ярославле.
  С июля 1922 по май 1926 работал в Н.Новгороде. В июле-декабре 1922 нач. эвакопункта.
  С декабря 1922 по декабрь 1924 пред. месткома, секретарь партячейки Всесоюзного
  нефтяного синдиката ВСНХ СССР (Нефтесиндикат). С декабря 1924 по май 1926 ответинструктор,
  зав. ОРГО горрайкома ВКП(б). С мая 1926 по октябрь 1927 ответсекретарь Сергачского
  укома ВКП(б) в г. Сергач Нижегородской губ. В октябре 1927 - июле 1929 работал в
  Н.Новгороде. С октября 1927 по апрель 1928 ответсекретарь горрайкома ВКП(б), в апреле
  1928 - июле 1929 зав. деревенским отделом губкома ВКП(б). С сентября 1929 по март
  1931 слушатель курсов м.-л. при ЦК ВКП(б) в Москве. С марта 1931 работал в Н.Новгороде
  (Горьком). В марте-декабре 1931 зав. отделом кадров, с декабря 1931 по январь 1934
  секретарь по рабочему снабжению крайкома ВКП(б). С февраля по декабрь 1934 зав.
  сов.-торг. отделом, с декабря 1934 по июнь 1937 2-й секретарь Горьковского крайкома
  ВКП(б). С июня 1937 по июнь 1938 пред. Горьковского облисполкома. Депутат ВС СССР
  1-го созыва. Арестован в Москве в гостинице "Националь" 3 июня 1938. Политбюро ЦК
  ВКП(б) 26 июля 1938 по списку санкционировало применение ВМН (1 кат.). ВКВС СССР
  29 июля 1938 за участие в КРТО по ст.ст. 58-7, 58-8, 58-9, 58-11 УК РСФСР приговорен
  к расстрелу, в тот же день расстрелян. Захоронен в Московской обл. в окрестностях
  пос. Коммунарка. В ЗАГС сообщено, что Буров умер, отбывая наказание, 17 декабря
  1945. Решением Горьковского обкома ВКП(б) 23 августа 1938 исключен из ВКП(б) "как
  враг народа, арестованный органами НКВД". Реабилитирован определением ВКВС СССР
  9 мая 1956.
