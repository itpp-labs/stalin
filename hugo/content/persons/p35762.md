---
title: ФРЕШЕР Евгений Эдуардович
description: Род. 17 октября 1890 в с. Розенталь (Школьное) Мелитопольского уезда
  Таврической губ. в семье рабочего мельницы. Немец. Чл. РКП(б) с марта 1919. В 1901
  окончил два класса нач. шк. в Царицыне. С октября 1901 по июль 1914 ученик, колбасник,
  пом. мастера консервно-колбасной ф-ки Киссельмана в Царицыне. С июля 1914 по август
  1915 рядовой 226-го пех. полка 105-й пех. дивизии Царской армии на Австро-Германском
  фронте. В августе-ноябре 1915 на излечении в военном госпитале в Киеве. С ноября
  по декабрь 1915 рядовой зап. полка в г. Новоград-Волынский. С декабря 1915 по июль
  1916 рядовой 23-го Туркестанского полка на Кавказском фронте. С июля по ноябрь 1916
  на излечении в военном госпитале в Баку. В ноябре 1916 - феврале 1918 рядовой 228-го
  зап. полка в Тифлисе. В феврале-июле 1918 рабочий-колбасник консервно-колбасной
  ф-ки Киссельмана и в партизанском отряде в Царицыне. С июля по декабрь 1918 красноармеец
  особого Московского полка X-й Армии на Южном фронте. С декабря 1918 по июль 1919
  ответинструктор по заготовке продуктов продкомарма1 X-й Армии в Царицыне и г. Камышин
  Саратовской губ. С июля по декабрь 1919 на излечении в военном госпитале в Камышине
  и Саратове. С декабря 1919 по апрель 1921 служил и работал в Царицыне. В декабре
  1919 - марте 1920 красноармеец военной комендатуры X-й Армии. С марта по сентябрь
  1920 секретарь парткома упр-я внутренних дел. С сентября 1920 по апрель 1921 ответсекретарь
  2-го Царицынского райкома ВКП(б). С апреля 1921 по сентябрь 1923 секретарь Оргбюро
  губкома ВКП(б), ответсекретарь Николаевского укома ВКП(б) Царицинской (Сталинградской)
  губ. С сентября 1923 по июнь 1928 ответсекретарь 1-го райкома ВКП(б), зав. ОРГО
  Сталинградского губкома ВКП(б). С июня 1928 по июнь 1932 работал в Саратове. В июне
  1928 - сентябре 1930 зав. ОРГО Нижне-Волжского крайкома ВКП(б). С сентября 1930
  по май 1931 пред. крайсовпрофа. С мая 1931 по июнь 1932 4-й секретарь Нижне-Волжского
  крайкома ВКП(б). С июня 1932 по январь 1934 1-й секретарь обкома ВКП(б) АССР НП
  в Энгельсе. С января по декабрь 1934 2-й секретарь Саратовского крайкома ВКП(б).
  С декабря 1934 по январь 1936 1-й секретарь обкома ВКП(б) АССР НП. С января 1936
  по январь 1937 пред. Саратовского крайисполкома. С января по июль 1937 1-й секретарь
  обкома ВКП(б) АССР НП. Снят с работы и исключен из ВКП(б). Арестован 22 июля 1937
  по делу "о подпольной националистической фашистской организации в АССР НП". Партдокументы
  погашены обкомом ВКП(б) НП в октябре 1937 как на исключенного. Политбюро ЦК ВКП(б)
  22 декабря 1937 и 10 июня 1938 по списку санкционировало применение ВМН (1 кат.).
  СССР в Москве 20 июня 1938 ВКВС по обвинению в шпионаже, террористическом акте,
  диверсии и АСД приговорен к расстрелу, в тот же день расстрелян. Реабилитирован
  определением ВКВС СССР 27 октября 1956.
