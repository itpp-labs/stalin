title: ВЕНЦОВ-КРАНЦ Семен Иванович
description: "Род. в 1897, еврей, обр.: Иркутское военное училище, 1921-1922 учился\
  \ в Военной Академии РККА, член ВКП(б) с 1918. 1919-1920 военком 1-й стрелковой\
  \ бригады 34-й стрелковой дивизии, 1920-1921 командир Московской территориальной\
  \ бригады, 1921 начальник штаба особой кавалерийской группы Уборевича, 1922-1923\
  \ командир батальона 52-го стрелкового полка 18-й стрелковой дивизии, 1923-1924\
  \ адъюнкт Военной Академии РККА по кафедре гражданской войны, 1924-1926 начальник\
  \ организационно-мобилизационного управления Штаба РККА, 1926-1928 для особых поручений\
  \ при председателе РВС СССР, 1928-1929 командир 15-го стрелкового полка 5-й стрелковой\
  \ дивизии, 1929-1931 помощник начальника Штаба РККА – начальник 2-го управления\
  \ Штаба РККА, 1931-1932 начальник штаба Белорусского ВО, 1932-1933 начальник штаба\
  \ Московского ВО, 1933 начальник Управления делами Наркомата по военным и морским\
  \ делам и РВС СССР, 1933-1936 военный атташе во Франции, Награжден двумя орденами\
  \ Красного Знамени., комдив(26.11.35), 1936-1937 командир 62-й Туркестанской стрелковой\
  \ дивизии \n  Арестован 11.06.1937. Приговор: ВК ВС СССР, 08.09.1937 – ВМН. Расстрелян\
  \ 08.09.1937. \n  Реабилитирован 30.06.1956"
