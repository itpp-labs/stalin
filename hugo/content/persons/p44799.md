title: ТОРОЩИН Павел Васильевич
description: "Род. 09.12.1894, Кировский кр., Шарангский р-н, высел. Преображенский,\
  \ русский, из крестьян, поручик, обр.: высшее, окончил Высшую школу штабной службы\
  \ комсостава РККА, 1921-1924 окончил Военную Академию РККА, член ВКП(б). Проживал:\
  \ г. Москва, ул. Покровка, 20/1-5. Командир стрелкового батальона Особого полка\
  \ Западного фронта, начальник общего отделения штаба 8-й армии, старший помощник\
  \ начальника штаба 16-й кавалерийской дивизии, 1924-1934 служба в УГПУ Ленинградского\
  \ округа, помощник начальника УПВО ОГПУ по Ленинградскому округу, 1934-1936 начальник\
  \ УПВО НКВД Северо-Кавказского края, награжден знаком “Почетный чекист”, 1936-1938\
  \ командир отдельной мотомеханизированной дивизии особого назначения НКВД СССР им.Дзержинского,\
  \ комбриг \n  Арестован 29.07.1938. Обв.: участие в к.-р. тер.организации. Приговор:\
  \ ВК ВС СССР, 28.01.1940 – ВМН. Расстрелян 29.01.1940. \n  Реабилитирован 06.10.1956"
