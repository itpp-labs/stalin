description: "Род. 15.6.1893, Новгородская губ., Кадуйский р-н, д. Калинниково, русский,\
  \ из крестьян, унтер-офицер царской армии., обр.: среднее, Высшие академические\
  \ курсы РККА, член ВКП(б) с 1919. 1918-1921 заместитель командующего войсками Туркестанской\
  \ республики, главком войск Туркестанской республики, начальник 3 Туркестанской\
  \ стрелковой дивизии, командующий Бухарской группой войск Туркестанской республики,\
  \ 1921-1922 командир 2-й стрелковой дивизии, 22-й стрелковой дивизии, 9-й стрелковой\
  \ дивизии, 1922-1927 командир 15-го стрелкового корпуса, 9-го стрелкового корпуса,\
  \ 2-го стрелкового корпуса, 11-го стрелкового корпуса, помощник командующего Московским\
  \ ВО, Северо-Кавказским ВО, 1927-1931 командующий Северо-Кавказским ВО, 1931-1935\
  \ командующий Ленинградским ВО, 1935-1937 командующий Московским ВО, Награжден двумя\
  \ орденами Красного Знамени (1920, 1921), командарм 1-го ранга (1935), 1937-1938\
  \ командующий войсками Белорусского ВО, член Военного Совета при наркоме обороны\
  \ СССР, член ЦИК СССР. \n  Арестован 07.01.1938. Обв. в участии в к.-р. организации\
  \ и военно-фашистском заговоре. Приговор: ВК ВС СССР, 29.07.1938 – ВМН. Расстрелян\
  \ 29.07.1938. \n  Реабилитирован 26.11.1955"
