title: РАЕВСКИЙ Артем Мартиянович
description: "член ВКП(б) с 1910, окончил сельскохозяйственное училище в Одессе. В\
  \ 1913 году вступил вольноопределяющимся в русскую армию. Революцию встретил в госпитале,\
  \ затем командовал крупными соединениями. После демобилизации в 1924 закончил институт\
  \ Красной профессуры, преподавал политическую экономию. С 1926 по 1933 годы был\
  \ профессором Академии Наркомвнешторга, вел там курс политэкономии. \n  Когда подбирали\
  \ кадры для политотделов МТС, он по рекомендации А. Микояна, у которого некогда\
  \ работал референтом, уехал из столицы на Сахалин в качестве начальника Сахалинснаба."
