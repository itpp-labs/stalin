СЕМЕНОВ Григорий Иванович

Род. в 1891, Юрьев, русский, обр.: высшее, член ВКП(б). Проживал: Москва, ул. Усачева, д. 29, кв. 356. Сотрудник Гл. разведывательного управления РККА.

Реабилитирован ВК ВС СССР 22.08.1961
Источник: Москва, расстрельные списки - Донской крематорий
