ПОЛЯНСКИЙ Никифор Александрович

1894 г.р., русский, член ВКП(б) с 1916, командир 82 СД УРАЛВО, комбриг (26.11.1935). Награды: орден Красного Знамени 09.01.1922.
Приговор: 09.08.1938 - ВМН, расстрелян 09.08.1938, Свердловск.
Реабилитирован 01.09.1956
Источник: Сведения о репрессированных военнослужащих, предоставленные А.Н.Жуковым

Род. в 1894, Астраханская губ., с. Светлый Яр, русский, обр.: высшее, член ВКП(б). Военнослужащий, командир дивизии
Обв. в принадлежности к контрреволюционной организации. Приговор: выездная сессия ВК ВС СССР, 09.08.1938 – ВМН. Расстрелян 09.08.1938.
