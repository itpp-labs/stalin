Гаранин Андрей Григорьевич
род. 1895 г.Мервь Туркменская ССР,, г.Починки Горьков.края, русский, обр.: высшее, в 1935г. окончил Мос. экономический ин-т инженеров транспорта МЭМИИТ, из крестьян-бедняков, член ВКП(б) с 1917г., нач.колесного цеха Мос. пассажирского вагоно-ремонтного з-да в Лианозове, ранее работал на з-де Войтовича, прож.: пос. Лианозово Ярослав.ж.д., Ярославская ул., д.227. Арест: 17.08.37г. ДТО ГУГБ НКВД Яр.ж.д.. Обв.: 58-7,8,11 (в 1936г. был направлен в длительную командировку в Германию, г.Ротибер для закупки и приемки станков. Находясь в Германии, по сомнительно дешевой цене закупил для себя несколько отрезов материала,, есть подозрение, что эту скидку ему устроила фирма в целях его расположения к себе. "Дважды задерживался гестапо".). Приговор: выездная сессия ВК ВС СССР в Ярославле, 30.12.37г. - ВМН, конфискация. Расстрелян 31.12.37г., г.Ярославль. Реабилитация: ВК ВС СССР 02.08.57г.
