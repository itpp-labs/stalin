Симонов Михаил Дмитриевич
Род. в 1890, Ленинград, русский, обр.: среднее. Проживал: Актюбинская обл., Темирский р-н, Темир.
Арестован УНКВД по Актюбинской обл. 20.07.1938. Обв. по ст. 58-2, 58-11, 58-8 УК РСФСР. Приговор: ВМН.
Реабилитирован Генеральной прокуратурой РК 15.06.1992 за отсутствием состава преступления
Источник: Сведения ДКНБ РК по Актюбинской обл.
