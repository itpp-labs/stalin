ТУЖИЛКИН Сергей Сергеевич

Род. в 1909, Татарстан, Казань, русский, обр.: среднее. Проживал: Северо-Казахстанская обл., Петропавловск. Плановик-экономист, хлебокомбинат.
Арестован УНКВД СКО. 15.02.1936. Обв. по ст. 58-10 УК РСФСР. Приговор: ОО НКВД СССР, 10.06.1936 – 3 года ссылки (высылки).
Реабилитирован Прокуратурой по Карагандинской обл. 15.02.1962 за отсутствием состава преступления
Источник: Сведения ДКНБ РК по Северо-Казахстанской обл.
