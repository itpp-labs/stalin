СКАЧКО-КРИВОЛУЦКАЯ Пелагея Васильевна

Род. в 1893, Донская обл., ст-ца Ярыжская, хут. Куликов, русская, обр.: 3 класса, член ВКП(б). Проживала: г. Иркутск. Продавец киоска Иркутского «Пивтреста»
Арестована 27.05.1937. Обв. по ст. 58-1а, 2, 11, 7, 8. Приговор: ВК ВС СССР, 05.06.1938 – ВМН. Расстреляна 22.06.1938.
Реабилитирована 29.05.1959
Источник: Книга памяти Иркутской обл.
