ЩЕКИН Василий Федорович

Род. 23.12.1887, д.Жданово Нагорьев.р-на МО (Яросл.обл.), русский, из крестьян-бедняков, обр. низшее, член ВКП(б) 1926-37 г.г., исключен из ВКП(б) как эсер в 1917-18 г.г., инспектор Спецвагонов Перовского ВРЗ НКПС, прож.: Шереметьево, 2-й Шереметьевский проезд, 38а (плат.Плющево, 2 Железнодор. проспект, 38а)
Арестован 15.01.38 г. ДТО ГУГБ НКВД Лен.ж.д. Обв. по ст.17-58-8, 58-9, 11. Приговор: ВК ВС СССР, 26.09.1938 - дело возвращено на доследование. ДТО НКВД Лен.ж.д.05.11.38 г.:"дело представить ВК ВС СССР для пересмотра". 29.12.38г. дело прекращено.
Источник: Сведения М.И.Губиной, извлеченные из документов ГА РФ
