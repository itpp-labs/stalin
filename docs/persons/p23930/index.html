АБДУЛЛАЕВ Сейфулло

Список от 3 февраля 1938г. [Москва-центр] - 1-я категория - РГАСПИ, ф.17, оп.171, дело 414, лист 378

7-й фонд ЦА ФСБ

р.1899

Дата расстрела: 08.02.1938

АБДУЛЛАЕВ Сайфулла (Сайфулло).

Род. в октябре 1899 кишлаке Хорог Памирского окр. в в семье сел. старосты. Таджик. Чл. РКСМ с 1922. Канд. в чл. РКП(б) с сентября 1921, чл. РКП(б) с января 1922. В 1914 окончил четырехклассную русско-туземную шк. в Хороге. С апреля 1914 по август 1916 крестьянин в хозяйстве матери в Хороге. С августа 1916 по июнь 1917 переводчик хозчасти военного поста в кишлаке Ишкашим (Памир). С июля 1917 по апрель 1926 работал в Хороге Автономной Горно-Бадахшанской обл. ТаджАССР. В июле 1917 - марте 1918 крестьянин в хозяйстве матери. С марта 1918 по ноябрь 1919 переписчик уездн. ревкома. С декабря 1919 по ноябрь 1920 пом. нач. партизанского отряда. С 1920 по 1921 красноармеец. В сентябре 1921 - сентябре 1922 на чекистской работе в особом отделе Памирского отряда войск ВЧК. С сентября 1922 по январь 1923 ответсекретарь волисполкома. С января 1923 по октябрь 1924 чл. чрезвычайной военно-полит. тройки. С октября 1924 по октябрь 1925 пред. Памирского обл. ревкома. В октябре 1925 - апреле 1926 зав. ОРГО Памирского партбюро ЦК КП(б) Узбекистана. С мая 1926 по сентябрь 1928 зав. АПО обкома ВКП(б) в г. Дюшамбе. С сентября 1928 по март 1929 инструктор Мотовилихинского райкома ВКП(б) в г. Мотовилиха Уральской обл. С апреля 1929 по январь 1937 работал в Дюшамбе (Сталинабаде). В апреле 1929 - сентябре 1930 ответсекретарь Гиссарского окружкома КП(б) Таджикистана. С октября 1930 по апрель 1931 зав. отделом кадров ЦК КП(б) Таджикистана. С мая 1931 по май 1932 нарком земледелия ТаджССР. С мая 1932 по апрель 1934 нарком РКИ ТаджССР и пред. ЦКК КП(б) Таджикистана. С мая 1934 по июль 1936 нарком земледелия ТаджССР. С июля 1936 по январь 1937 2-й секретарь ЦК КП(б) Таджикистана. С января 1937 слушатель ВШПО при ЦК ВКП(б) в Москве. Перед арестом нарком мест. пром-сти ТаджССР. Арестован 12 сентября 1937 НКВД ТаджССР. Партдокументы погашены ЦК КП(б) Таджикистана в октябре 1937 как на исключенного. Политбюро ЦК ВКП(б) 3 февраля 1938 по списку санкционировало применение ВМН (1 кат.). ВКВС в Москве 7-8 апреля 1938 по ст.ст. 58-1а, 58-7, 58-8, 58-11 УК РСФСР приговорен к расстрелу с конфискацией имущества, в тот же день расстрелян. Сыну Абдуллаева в октябре 1956 сообщено, что его отец умер, отбывая наказание 21 ноября 1943. Реабилитирован определением ВКВС СССР 29 декабря 1956.

Источник: Филиппов С.Г. Территориальные руководители ВКП(б) в 1934-1939 гг.
