ВОЙТЕНКО Павел Макарович

Род. в 1892, Украина, Донецкая обл., Макеевский р-н, с. Нижняя Кромка, украинец, член УКП, потом КП(б)У. Проживал: Украинская ССР, г. Харьков, Рыбасовская, 30. Инженер-электрик, ХТЗ
Арестован 23.02.1938. Обв. по ст. 54-8-9-11 ("участник антисоветской террористической диверсионной, повстанческой националистической организации"). Приговор: ВК ВС СССР, 21.10.1938 – ВМН. Расстрелян 21.10.1938, г.Харьков.
Реабилитирован 23.08.1962
Источник: База данных о жертвах репрессий Харьковской обл. (Украина)

Род. в 1892, Донецкая обл., Макеевский р-н, с. Н.-Крынка. Проживал: г. Харьков, Рубановская, 30. Инж.-электрик на Литейно механич.з-де
Арестован УНКВД по Харьков.обл. 23.02.1938. Обв. по ст. 54-9, 8, 11 УК УССР. Приговор: ВК ВС СССР, 21.10.1938 – ВМН с конфискацией имущества. Расстрелян 21.10.1938, г.Харьков.
Реабилитирован ВК ВС СССР 12.07.1962
Источник: УСБУ по Харьковской обл.
