Рощин Михаил Кириллович
(р. 1905) инспектор нархозучета. Суть дела: Рощин, проводя Всесоюзную перепись населения, выполнял указания врагов народа Лебедева, Гудымовича и Пинчука по вредительскому проведению переписи. С целью срыва избирательной кампании в Верховный Совет СССР, Рощин по указанию врага народа Лебедева провел разбивку избирательных участков неправильно, пользовался списками всесоюзной переписи, которые были отменены как вредительски составленные… Рощин является ставленником врага народа Лебедева, который таскал его за собой для вредительской работы. Парторганизация Горсовета Рощина исключила из партии. Постановили: Решение парторганизации утвердить, Рощина Михаила Кирилловича за вредительскую работу по составлению списков избирателей, развал работы в нархозучете, за связь с врагом народа Лебедевым из рядов ВКП(б) исключить.
3 января 1938г.
