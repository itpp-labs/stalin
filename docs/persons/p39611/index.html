БОГОМОЛОВ Степан Федорович

Род. в 1899, Пензенская обл., Больше-Вьясский р-н, с. Ильино, русский, из крестьян, обр.: начальное, член ВКП(б). Проживал: Пензенская обл., Н. Ломовский р-н, г. Н. Ломов. Н.Ломовский РК ВКП(б), секретарь
Арестован Н.Ломовским РО УНКВД по Тамбовской обл. 11.03.1938. Обв. по ст. 17-58-7-8-11 - являлся активным а/с право-троцкистской и диверсионно-вредительской организации, проводил вредительскую работу. Приговор: Верховный суд СССР, 25.10.1938 – ВМН. Расстрелян 25.10.1938.
Реабилитирован Верховным судом СССР 12.07.1957 за отсутствием состава преступления
Источник: База данных о жертвах репрессий Пензенской обл.
