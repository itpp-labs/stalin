Дата сессии ВК ВС: 10.07.1938

Дата расстрела: 10.07.1938

КОРМЩИКОВ Григорий Михайлович

Род. в 1898 станции Иланская Томской ж.д. в Анцирской волости Канского уезда Енисейской губ. в семье рабочего-кочегара. Русский. Чл. РКСМ с 1920. Чл. РКП(б) с марта 1920. В 1914 окончил ж.-д. шк. на ст. Иланская., в 1916 четырехклассное высш. нач. уч-ще в Канске, в 1919 три класса землемерного уч-ща в Красноярске. С декабря 1913 по июль 1914 ученик частной фотографии в Канске. С июля 1914 по август 1915 рабочий у разных лиц на ст. Иланская. С августа 1915 по август 1916 учащийся высш. нач. уч-ща в Канске. С августа 1916 по август 1919 учащийся землемерного уч-ща в Красноярске. С августа 1919 по июль 1920 работал на ст. Иланская. В августе - декабре 1919 на поденной работе. С декабря 1919 по март 1920 болел, не работал. С марта по июль 1920 зав. информ. отделом участкового политотдела ж.-д. трансп. В июле-ноябре 1920 зав. учполитотделом ж.д., с ноября 1920 по апрель 1922 зав. культотделом учкпрофсожа на ст. Боготол Томской губ. В апреле-декабре 1922 ответсекретарь месткома ж.-д. трансп. на ст. Иланская. С декабря 1922 по июль 1923 ответсекретарь линейного профсоюза железнодорожников (линпрофсож) в Красноярске. С июля 1923 по октябрь 1925 зав. культотделом дорожного комитета профсоюза работников ж.-д. транспорта (дорпрофсож) в Иркутске. С октября 1925 по октябрь 1926 секретарь Зиминского укома ВКП(б) в г. Зима Иркутской губ. С октября 1926 по октябрь 1927 ответинструктор Иркутского окружкома ВКП(б) в Иркутске. С октября 1927 по февраль 1928 пред. учкпрофсожа на ст. Иннокентьевская Иркутской губ. В феврале-ноябре 1928 секретарь Заларинского райкома ВКП(б) в с. Тагна, с ноября 1928 по сентябрь 1929 в с. Залари Иркутского окр. С сентября 1929 по март 1934 работал в Иркутске. В сентябре 1929 - сентябре 1930 зав. АМО Иркутского окружкома ВКП(б). С сентября 1930 по июль 1932 зам. АМО Восточно-Сибирского крайкома ВКП(б). В июле-декабре 1932 пред. краев. Колхозсоюза. С декабря 1932 по апрель 1933 зам. зав. крайЗУ. С апреля 1933 по март 1934 зав. ОРГО и ответсекретарь Восточно-Сибирского крайисполкома. В марте-декабре 1934 пред. Оргкомитета ЦИК СССР по Читинской обл. С марта по август 1934 одновременно зам. секретаря Оргбюро Восточно-Сибирского кайкома ВКП(б) по Читинской обл. С декабря 1934 по август 1935 нач. райполитотдела на ст. Ерофей Павлович Забайкальской ж.д. С августа 1935 по июнь 1936 зам. нач. политотдела Северо-Кавказской ж.д. в г. Орджоникидзе СКК. С ноября 1936 по июнь 1937 зам. нач. политотдела по кадрам Московско-Киевской ж.д. в Калуге. В июне 1937 снят с работы как "соучастник к.р. троцкистской банды" и арестован. Политбюро ЦК ВКП(б) 10 июня 1938 по списку санкционировало применение ВМН (1 кат.). ВКВС СССР 10 июля 1938 приговорен к расстрелу, в тот же день расстрелян.

Источник: Филиппов С.Г. Территориальные руководители ВКП(б) в 1934-1939 гг.

Кормщиков Георгий Михайлович

Приговор: ВК ВС СССР, 10.07.1938 – ВМН
