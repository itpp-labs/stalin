ЛЕВЧЕНКО Петр Павлович

Род. в 1895. Род занятий: до ареста начальник продовольственного снабжения Тбилисского военного училища. По званию интендант 3 ранга. В РККА с 1925 г. Бывший офицер царской армии. Служил в меньшевистской милиции.
Осужден Тройкой при НКВД ГССР 09.11.1937. Мера наказания: расстрел с конфискацией личного имущества. Дата расстрела: 11.11.1937
Источник: По материалам Архива внутренних дел Грузии подготовлено Институтом развития свободы информации (IDFI), Грузия
