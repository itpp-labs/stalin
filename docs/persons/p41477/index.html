ШИМАНОВ Иван Иванович

Первый директор Кинешемского медицинского училища профессиональный революционер, провел большую работу по организации училища, по набору штата. Возглавлял училище недолго, через несколько месяцев оставил работу по состоянию здоровья. Фотографии не сохранилось.
