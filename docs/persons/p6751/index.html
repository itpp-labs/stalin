ПОТАПЕНКО Роман Яковлевич

Список от 25 августа 1937 г. [Центральный аппарат УГБ НКВД УССР] - 1-я категория - РГАСПИ, ф.17, оп.171, дело 410, лист 287

7-й фонд ЦА ФСБ

Дата сессии ВК ВС: 01.09.1937

Дата расстрела: 01.09.1937

Потапенко Роман Яковлевич

Род. в 1897 в г. Тараща Таращанского уезда Киевской губ.в семье столяра. Еврей. Чл. РКП(б) с сентября 1918. В 1912 окончил четырехклассное еврейское уч-ще в Тараще, в 1920 1-е Московские пулеметные курсы комсостава. С сентября 1912 по сентябрь 1917 наборщик типографии Рахлиса в Тараще. С октября 1917 по май 1918 безработный в Тараще. В июне-августе 1918 наборщик типографии ревсовета в Н.Новгороде. С сентября 1918 по февраль 1919 красноармеец запасного полка в Н.Новгороде. С марта 1919 по апрель 1920 курсант 1-х Московских пулеметных курсов. В мае 1920 - ноябре 1921 нач. пулеметной команды 251-го полка 24-й дивизии РККА. С декабря 1921 по апрель 1934 работал в Харькове. В декабре 1921 - октябре 1924 наборщик типографии штаба УВО. С ноября 1924 по январь 1927 секретарь парткома типографии штаба УВО. С февраля 1927 по июль 1929 зае. ОРГО Журавлевского райкома КП(б)У. В августе 1929 - июле 1930 зам. зав. ОРНО Харьковского окружкома КП(б)У. С августа 1930 по апрель 1934 секретарь парткома Харьковского тракторного з-да. С мая 1934 по октябрь 1936 работал в Киеве. В мае 1934 - марте 1935 зам. секретаря Киевского горкома КП(б)У. С марта 1935 по октябрь 1936 2-й секретарь Киевского горкома КП(б)У. Одновременно с 1935 по октябрь 1936 зав. отделом парткадров Киевского горкома КП(б)У. С октября 1936 по июнь 1937 слушатель ВШПО в Москве. В мае-июле 1937 2-й секретарь Черниговского горкома КП(б) Украины. Арестован 7 июля 1937 НКВД УССР. Политбюро ЦК ВКП(б) 25 августа 1937 по списку санкционировало применение ВМН (1 кат.). Выездной сессией ВКВС СССР в Киеве 1 сентября 1937 по ст.ст. 54-8, 54-11 УК УССР приговорен к расстрелу. Расстрелян 2 сентября 1937. Реабилитирован определением ВКВС СССР 12 ноября 1955.

Источник: Филиппов С.Г. Территориальные руководители ВКП(б) в 1934-1939 гг.
