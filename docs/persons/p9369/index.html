Борщ Григорій Минович

1899 р., с. Кушнарівка Верхньодніпровського р-ну Дніпропетровської обл., українець, освіта початкова, член КП(б)У з 1921 р. по 1937 р. Проживав у м. Хмельницькому, секретар Проскурівського окружного комітету КП(б)У. Заарештований 02.08.37. Звинувачення: член контрреволюційної організації. Військколегією Верховного Суду СРСР 16.10.37 засуджений до розстрілу. Вирок виконаний. Реабілітований військколегією Верховного Суду СРСР 24.08.57. (П-4995, архів УСБУ).

Источник: Реабілітовані історією : Хмельницька область
