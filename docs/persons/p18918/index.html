ШУЛЬМЕЙСТЕР Константин Адамович

партийный и советский деятель АССР Немцев Поволжья, в 1930-е годы нарком просвещения АССР НП. В 1936 - член комиссии по подготовке проекта новой Конституции АССР Немцев Поволжья. С апреля 1937 - первый секретарь Бальцерского канткома ВКП(б) АССР НП. Через несколько месяцев был арестован. 19 августа 1937 исключен из партии. Дальнейшая судьба неизвестна.
Александр Шпак
