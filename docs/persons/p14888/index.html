Савельев Дмитрий Андреевич
1905 г.р., русский, м.р. с.Благовещенское, Владимирский р-н, Иваново-Промышленная обл., м.п. г.Калинин. Арестован 08.05.1937. Осужден 05.10.1940 ОСО при НКВД СССР за разглашение гостайны на 8 лет лишения свободы. Прибыл 22.12.1940 из тюрьмы г.Калинина. Освобожден 08.05.1945. Реабилитирован 26.11.1956 военным трибуналом МВО.
Источник: Книга памяти Республики Коми
