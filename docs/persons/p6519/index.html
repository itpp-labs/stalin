Раудмец Иван Иванович

Род. в 1892, эстонец, из крестьян, капитан, обр.: военное училище, 1922–1924 окончил Высшие академические курсы РККА, член ВКП(б) с 1927. Командир батальона в 4-й Петроградской стрелковой дивизии (15.III.-1.VI.18 г.), помощник командира 5-го стрелкового полка (1-16.VI.18 г.), командир 14-го Нарвского коммунистического полка (18.VI-25.XI.18 г.), 156-го (25.XI.18 г.-1.11.19 г.), 157-го (1-10.11.19 г.) стрелковых полков, командовал Шенкурским районом (10.11.-18.IV.19 г.), 2-й бригадой 18-й стрелковой дивизии (18.IV.-27.VI.19 г.), находился в распоряжении РВС 6-й армии (с 27.VI.19 г.), командир 2-й бригады 52-й стрелковой дивизии (с 3.VIII.19 г.), начальник 52-й (22.IX.19 г.-12.V.20 г., 4-13.VI.20r.) и 15-й (3.VIII.20г.-1.22г„ IX-XII.22 г.) стрелковых дивизий, командир 48-й стрелковой дивизии, 1924-1937 начальник снабжения Московского ВО, комендант Полоцкого УР, командир 99-й стрелковой дивизии, комендант Новоград-Волынского УР, награжден двумя орденами Красного Знамени, комдив, 1937 комендант Могилев-Ямпольского УР
Арестован 11.06.1937. Приговор: ВК ВС СССР, 09.09.1937 – ВМН. Расстрелян 09.09.1937.
Реабилитирован 04.08.1966

Источник: Чураков Д. Репрессированные военнослужащие Красной Армии
