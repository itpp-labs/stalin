АЙВАЗОВ Моисей Гаврилович

Список от 19 апреля 1938г. [Татарская АССР] - 1-я категория - РГАСПИ, ф.17, оп.171, дело 416, лист 84

7-й фонд ЦА ФСБ

Дата сессии ВК ВС: 09.05.1938

Дата расстрела: 09.05.1938

Айвазов Моисей Гаврилович

Род. в 1900 в Тифлисе в семье шапочника. Армянин. Чл. РКП(б) с января 1918. В 1916 окончил ремесленное уч-ще, в 1918 учился на средне-техн. курсах в Тифлисе, в 1935 на курсах м.-л. при ЦК ВКП(б) в Москве. С ноября 1918 по май 1920 на нелегальной работе, парторганизатор 3-го р-на в Тифлисе. С июня 1920 по ноябрь 1922 уполн. особого отдела 10-й армии РККА. С декабря 1922 по декабрь 1923 инструктор Заккрайкома РКП(б) в Тифлисе. С февраля 1924 по март 1927 зав. ОРГИ Книгосоюза, с марта 1927 по июль 1928 зав. райотделом заготовок Всероссийского центр. союза "Хлебоцентр" РСФСР в Москве. С августа 1928 по август 1929 пред. правл. окрполеводсоюза в Новосибирске. С сентября 1929 по июль 1933 работал в Москве. В сентябре 1929 - феврале 1931 чл. правл. облполеводсоюза. С февраля 1931 по ноябрь 1932 инструктор МК ВКП(б). С ноября 1932 по июль 1933 пом. зав. Отдела агитации и массовых компаний ЦК ВКП(б). С августа 1933 по декабрь 1934 зам. нач. политсектора НКЗ БССР в Минске. В феврале-октябре 1935 слушатель курсов м.-л. в Москве. С октября 1935 по апрель 1936 2-й секретарь Хакасского обкома ВКП(б). С мая 1936 по ноябрь 1937 1-й секретарь Ленинского райкома ВКП(б) в Казани. 30 ноября 1937 исключен из партии "как враг народа". Арестован 2 декабря 1937. Политбюро ЦК ВКП(б) 19 апреля 1938 по списку санкционировало применение ВМН (1 кат.). Выездной сессией ВКВС СССР в Казани 9 мая 1938 по ст.ст. 58-7, 58-8, 58-11 УК РСФСР приговорен к расстрелу с конфискацией имущества, в тот же день расстрелян. По другим данным сессия проходила 9-12 мая 1938, дата расстрела 12 мая 1938. Реабилитирован в марте 1956.

Источник: Филиппов С.Г. Территориальные руководители ВКП(б) в 1934-1939 гг.
