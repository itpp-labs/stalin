РОЗАНОВ Иван Иванович

7-й фонд ЦА ФСБ

Дата сессии ВК ВС: 25.12.1937

Дата расстрела: 26.12.1937

Списки

Список от 7 декабря 1937г. [Украинская ССР] - РГАСПИ, ф.17, т.5, оп.171, дело 413, лист 126

Розанов Иван Иванович
Род. в 1904, Бавлинский р-н, дер. Фоминовка, русский, член ВКП(б). Ст. лейтенант, ком. роты, 296 стр. полк
Арестован 02.08.1937. Обв. по ст. 54-1б, 54-11, 54-8 УК УССР. ("участник военно-фашистского заговора"). Приговор: ВК ВС СССР, 25.12.1937 – ВМН с конфискацией имущества. Расстрелян 26.12.1937, в г.Киев.
Реабилитирован 19.01.1967
Источник: Книга памяти Республики Татарстан
