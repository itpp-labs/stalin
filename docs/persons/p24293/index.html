КОРЧЕМНЫЙ Елим Израилевич

Род. в 1903, г. Одесса, еврей. Проживал: Дальневосточный кр., Советский р-н, б/х Нанте. Начальник электромонтажного отдела строительства [ГЭС "ДЭСНА" (Дальневосточная электростанция спец. назначения), Советско-Гаванский р-н, пос. Майский]
Арестован 01.02.1937. Обв. по ст.ст.58-8, 58-11 УК РСФСР. Приговор: выездная сессия ВК ВС СССР, 11.03.1938 – ВМН. Расстрелян 11.03.1938, г.Хабаровск.
Реабилитирован ВК ВС СССР 02.11.1957
