БЕРЛИН Борис Михайлович

Род. в 1899, Смоленская губ., Горецкий уезд, с. Ляды [?], еврей, обр.: среднее. Проживал: Алма-Ата. Уполномоченный по Казахстану, заготзерно.
Арестован НКВД Каз.ССР. 04.11.1937. Обв. по ст. 58-8, 58-9, 58-11 УК РСФСР. Приговор: Верховный суд СССР, 27.02.1938 – ВМН.
Реабилитирован Верховным судом СССР 02.10.1957 за отсутствием состава преступления
Источник: Сведения ДКНБ РК по г.Алматы
