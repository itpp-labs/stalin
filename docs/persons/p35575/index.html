1904 г.р., урож. БМАССР, Хоринский, населенный пункт В-Кижингинский. Бурят, грамотный, проживал: БМАССР, Улан-Удэ.
Арестован 26.11.1937. Дело рассмотрено Верховным Судом СССР 02.06.1938, по ст.ст. 58-1а, 58-2, 58-7, 58-8, 58-11. Решение по делу: ВМН.
