ИСКАКОВ Гарибулла Мусабаевич

Род. в 1900, Восточно-Казахстанская обл., Чингизовский [?], казах, обр.: среднее специальное. Проживал: Алма-Ата. Директор, Казахский НИИ Марксизма-Ленинизма.
Арестован УНКВД по Алма-Атинской обл. 02.11.1937. Обв. по ст. 58-7, 58-8, 58-11 УК РСФСР. Приговор: выездная сессия ВК ВС СССР, 26.02.1938 – ВМН.
Реабилитирован ВК ВС СССР 19.05.1956 за отсутствием состава преступления
Источник: Сведения ДКНБ РК по г.Алматы
