ДЖАЛЕЛОВ Нигмат

Род. в 1900, Западно-Казахстанская обл., Денгизский р-н, аул Батахан, казах, обр.: неполное среднее. Проживал: Южно-Казахстанская обл. (Чимкентская), Кзылкумский р-н, село Коксай. Секретарь сельсовета, Коксайский сельсовет.
Арестован 16.10.1937. Обв. по ст. 58-2, 58-7, 58-8, 58-11 УК РСФСР. Приговор: ВК ВС СССР, 21.02.1938 – ВМН.
Реабилитирован ВК ВС СССР 10.07.1959
Источник: Сведения ДКНБ РК по Южно-Казахстанской обл.
