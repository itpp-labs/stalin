ОРАХЕЛАШВИЛИ Мамия (Иван) Дмитриевич

(1881-1937), грузинский политический деятель. С 1920 член Кавбюро ЦК РКП(б), председатель (секретарь) ЦК КП(б), председатель СНК Грузии, СНК ЗСФСР. С 1923 заместитель председателя СНК СССР. С 1926 секретарь Заккрайкома ВКП(б), председатель СНК ЗСФСР. С 1932 на научной работе. Репрессирован, реабилитирован посмертно.
Источник: Большой энциклопедический словарь

Род. в 1883, грузин. Место проживания: г. Москва, ул. Грановского №3, кв.83. Род занятий: До ареста заместитель директора Института Маркса-Энгельса, Ленина при ЦК КП(б).
Осужден Тройкой при НКВД ГССР 03.12.1937. Мера наказания: расстрел с конфискацией личного имущества. Дата расстрела: 11.12.1937
Источник: По материалам Архива внутренних дел Грузии подготовлено Институтом развития свободы информации (IDFI), Грузия
