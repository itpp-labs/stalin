Джаарбеков (Джарбеков) Иван Николаевич
Род. в 1899. Армянин. Из служащих. Интендант 3 ранга (11.04.1936). В РККА с 1927. Член компартии с 1925. Окончил специальное (агентурное) отделение РКУКС при РУ штаба РККА (1934). В распоряжении РУ штаба РККА, начальник пограничного разведывательного пункта ГО штаба Кавказской Краснознаменной армии — Закавказского ВО (март 1932 — август 1937), из них начальник ПРП № 3 — Эривань (июль 1934 — август 1937). Уволен со службы в РККА 23.08.1937 как арестованный органами НКВД.
Источник: Алексеев М.А., Колпакиди А.И., Кочик В.Я. Энциклопедия военной разведки. 1918-1945 гг. М., 2012

Джаарбеков Иван Николаевич
Род. в 1889, Иран, армянин. Место проживания: г. Ереван, Армянская ССР. Род занятий: военнослужащий.
Осужден ВК ВС СССР 03.10.1937. Мера наказания: расстрел. Дата расстрела: 03.10.1937
Источник: По материалам Архива внутренних дел Грузии подготовлено Институтом развития свободы информации (IDFI), Грузия
