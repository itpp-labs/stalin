О проекте

СЕНЮШКИН Федор Михайлович

р.1885 (1886) в Троицке, Пензен. губ., профсоюзный работник член Викжеля, счетовод Забайкальской железной дороги, в 1919 г. - член ВЦСПС. В 1924 г. - член Комиссии для политического контроля над работой киноорганизаций при Агитпропе ЦК РКП(б). С 1925 г. - член Художественного совета по делам кино при Главполитпросвете НКП РСФСР. Репресситрован.
Источник: Архив еврейской истории. Т.4. - М., 2007.
