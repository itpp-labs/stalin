Паршин Иван Павлович
род. 1905, с.Ембулатово, Буинский р-н, Татарская АССР, русский. Из рабочих. Образование высшее. Член ВКП(б) с 1926. Исключен 21.06.1936 решением бюро Балаханского РК АКП(б) как активный контрреволюционер троцкист-террорист. Восст. посмертно 14.10.1958 решением бюро ЦК КП Азербайджана.
Техник по бурению (буровой мастер) 1-й группы бурения треста "Лениннефть". Прож.: Аз.ССР, г.Баку.
Арестован 22.05.1936
Обвинение: ст.ст. 70,73 УК Аз.ССР - участник с 1935 троцкистской террор. группы на з-де им.Буденного (Левагин и др.), давший согласие на исполнение теракта, обучавшийся для этого стрельбе из револьвера.
Приговор: ВК ВС СССР, 03.07.1937 - ВМН с конфискацией имущества.
Реабилитирован ВКВС СССР 04.02.1956 за отсутствием состава преступления.
*Семье выдано свидетельство о смерти 1-ИЕ №032782 с датой смерти 02.01.1940.
Источники: Сталинский список от 27.02.1937 (Баку, Кат.1), 23.04.1937 (Баку, Кат.1)| Личное дело №39090 (АПД УДПАР, ф.6, оп.9, д.398)| Определение ВКВС СССР, 04.02.1956.
Источник: Правозащитный Центр Азербайджана
