Бурханов (Бурхонов) Назир
1903 г.р., м.р.: Нурота, узбек, из бедной крестьянской семьи, образование: с октября 1926 г. по июль 1929 г. учился на рабочем факультете им.Сталина в Москве, с октября 1929 г. до августа 1930 г. - в педагогическом институте им.А.Икромова в Самарканде, с августа 1930 г. по июль 1934 г. - в Московском текстильном институте, член ВКП(б) с 1922 г. (на другой странице - с 1923 г.?)
с октября 1936 г. первый секретарь партийного комитета Фрунзенского района г.Ташкента
