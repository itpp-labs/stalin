ТУХАРЕЛИ Георгий Александрович

1898 г.р., грузин, канд. в члены ВКП(б) с 1932, пом. командующего ЗАКВО по МТО, комдив (26.11.1935). Награды: орден Красной Звезды 16.08.1936 ; орден Красного Знамени 1925 ; орден Трудового Красного Знамени Азербайджанской ССР 1932 ; орден Трудового Красного Знамени Армянской ССР 1933.
Арестован 17.04.1937. Приговор: 12.07.1937 - ВМН, расстрелян 12.07.1937, Тбилиси.
Реабилитирован 09.01.1957
Источник: Сведения о репрессированных военнослужащих, предоставленные А.Н.Жуковым

Род. в 1891, Горийский р-н, грузин. Место проживания: г. Тбилиси, ул. Судебная № 33. Род занятий: до ареста помощник командующими войсками ЗакВО.
Осужден ВК ВС СССР 13.07.1937. Мера наказания: расстрел. Дата расстрела: 13.07.1937
Источник: По материалам Архива внутренних дел Грузии подготовлено Институтом развития свободы информации (IDFI), Грузия
