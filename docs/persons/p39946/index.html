ОРДАНОВ Александр Николаевич

Список от 12 сентября 1938г. [Ростовская область] - 1-я категория - РГАСПИ, ф.17, оп.171, дело 418, лист 197

Орданов Александр Николаевич
Род. в 1894, украинец, из рабочих, обр.: 1918 окончил Военно-инженерный техникум, член ВКП(б) с 1930. 1918-1919 командир взвода 8-го инженерного батальона, 1919-1920 помощник начальника разведотдела штаба 12-й армии, 1920-1921 помощник начальника штаба по разведке 45-й стрелковой дивизии, 1921-1922 начальник разведотдела штаба 3 кавалерийского корпуса, 1922-1923 начальник разведки 7-й кавалерийской дивизии, 1923 начальник разведки 14-го стрелкового корпуса, 1923-1924 для поручений разведотдела штаба Украинского ВО, 1924-1927 начальник разведки, помощник начальника оперативной части штаба 1-го кавалерийского корпуса, 1927-1930 помощник начальника 1-го (оперативного) отдела штаба Северо-Кавказского ВО, 1930-1933 начальник разведотдела штаба Северо-Кавказского ВО, 1933-1936 помощник начальника по разведке 1-го отдела штаба Северо-Кавказского ВО, майор (1935), 1936-1937 начальник разведотдела штаба Северо-Кавказского ВО, уволен из РККА в мае 1937г.
репрессирован.
Источник: Чураков Д. Репрессированные военнослужащие Красной Армии
