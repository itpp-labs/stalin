Шерлаимов Иван Фролович

Дата рождения: 1900
Год вступления в КПСС: 1926
Образование
сельская школа

март 1927 – июль 1930 — агент управления государственного страхования по Ординскому району
июль 1930 – июль 1931 — председатель Ординского райисполкома

сентябрь 1931 – июль 1932 — председатель Оханского райисполкома
