Сафиев Ахмед Сафарали-оглы

род. 1901, азербайджанец. Из крестьян. Член ВКП(б) с 1925. Исключен 30.09.1937 решением ПК при ПО 77-й Аздивизии. Подтверждено 05.03.1938 ПК при ПУ ЗакВО. Восст. посмертно 11.06.1957 решением бюро ЦК КП Азербайджана.
Интендант II ранга. Военком Закатальского РВК. Прож.: Аз.ССР, г.Закаталы
Арестован в 1937
Обвинение: ст.ст. 64,70,73 УК Аз.ССР - участник с 1935 буржуазно-нац-ской орг-ции в Аз.ССР, которая готовила вооруженное восстание против Советской власти (военно-фашистского заговора в 77-й Азерб. ГСД).
Приговор: ВК ВС СССР, 03.01.1938 - ВМН с конфискацией имущества.
Расстрелян 03.01.1938
Реабилитирован ВКВС СССР 27.10.1956 за отсутствием состава преступления.
Источники: Сталинский список от 07.12.1937 (Аз.ССР, Кат.1)| Личное дело №37525 (АПД УДПАР, ф.6, оп.9, д.461)| Определение ВКВС СССР, 27.10.1956| Определение ВТ ЗакВО, 18.06.1957 (упоминание).

Источник: Правозащитный Центр Азербайджана
