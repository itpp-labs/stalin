ВОРОНИН Николай Иванович

Род. в апреле 1916, г.Москва, русский, из рабочих, обр. семилетка, среднее техническое, механик двигателей внутр.сгорания, б/п, слесарь экспериментального цеха автозавода им.Сталина, прож.: Москва, Арбат, 36 - 3
Арестован 13.08.37 г. УГБ УНКВД МО. Обв. по ст.58-8, 11. Приговор: ВК ВС СССР, 04.09.1937 - 8 лет тюрьмы с поражением прав 3 года, конфискация имущества
Арестован. Обв.: принадлежность к троцкист.-террор. орг-ции. Приговор: ОСО МГБ СССР, 29.11.50 г. - ссылка на поселение в Красноярский край, г.Дудинка, контора водных перевозок
Реабилитация: ВК ВС СССР, 12.05.56
Источник: Сведения М.И.Губиной, извлеченные из документов ГА РФ
