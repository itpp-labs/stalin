Источник: Реабілітовані історією : Дніпропетровська область

МАТВЕЕВ Николай Иванович

Род. 29 ноября 1894 в Селищинских казармах (д.Селищи) Новгородского уезда и губ. в семье старшего писаря батарейной канцелярии. Русский. Чл. РКСМ с 1918. Чл. РКП(б) с августа 1919. В 1909 окончил двухклассную ЦПШ в Селищенских казармах Новгородской губ. С июня 1909 по август 1915 гравер по металлу на з-де Новицкого в С.-Петербурге (Петрограде). С августа 1915 по декабрь 1917 служил в Царской армии. В августе 1915 - июле 1916 рядовой 314-го Глуховского пех. полка. С июля 1916 по июнь 1917 разведчик, в июне-декабре 1917 ст. писарь 37-й арт. бригады. С декабря 1917 по ноябрь 1918 рабочий на Мало-Вишерском стекольном з-де братьев Курженковых в г. Малая Вишера Новгородской губ. С ноября 1918 по март 1919 стрелок ЧК по охране ж.д. на ст. Малая Вишера Николаевской ж.д. С марта 1919 по январь 1920 стрелок 99-го батальона 34-го Мало-Вишерского полка ж.-д. охраны РККА. В январе-ноябре 1920 переписчик 34-го Мало-Вишерского полка. С ноября 1920 по июнь 1921 делопроизводитель хозчасти 170-го стр. полка ВНУС на ст. Жлобин Московско-Белоруссско-Балтийской ж.д. С июня 1921 по май 1922 зав. ОРГО Рогачевского укома ВКП(б) в г. Рогачев Гомельской губ. С мая 1922 по апрель 1923 зав. ОРГО Почепского укома РКП(б) в г. Почеп Гомельской губ. С апреля 1923 по март 1925 секретарь Новозыбковковского укома ВКП(б) в г. Новозыбков Гомельской губ. В марте 1925 - августе 1926 зав. ОРГО Гомельского губкома ВКП(б). С августа 1926 по июль 1929 зав. ОРГО Брянского губкома ВКП(б). С июля 1929 по апрель 1930 ответсекретарь Брянского окружкома ВКП(б). С апреля 1930 по декабрь 1931 пред. облпрофсовета в Смоленске Западной обл. С декабря 1931 по октябрь 1933 пред. ЦК профсоюза сахарников в Москве. С октября 1933 по март 1937 2-й секретарь Днепропетровского обкома КП(б)У. С марта по июль 1937 1-й секретарь Запорожского горкома КП(б)У. Арестован 9 июля 1937 Запорожским горотделом НКВД. Решением бюро Днепропетровского обкома КП(б)У 16 июля 1937 снят с работы и исключен из ВКП(б) "как враг народа". Политбюро ЦК ВКП(б) 25 августа 1937 по списку санкционировало применение ВМН (1 кат.). Выездной сессией ВКВС СССР в Днепропетровске 14 сентября 1937 по ст.ст. 54-8 (теракт), 54-11(участие в КРО) УК УССР приговорен к расстрелу с конфискацией имущества. Расстрелян 15 сентября 1937. Реабилитирован определением ВКВС СССР 22 сентября 1956 и по ст.1 Закона Украины "О реабилитации жертв полит. репрессий" от 17 апреля 1991.

Источник: Филиппов С.Г. Территориальные руководители ВКП(б) в 1934-1939 гг.
