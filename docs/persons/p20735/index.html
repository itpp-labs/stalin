ЖАМБУЛОВ Абушахман

Список от 3 января 1938г. [Казахская ССР] - 1-я категория - РГАСПИ, ф.17, оп.171, дело 414, лист 110

7-й фонд ЦА ФСБ

Дата сессии ВК ВС: 09.03.1938

Дата расстрела: 09.03.1938

Жамбулов Абушахман

Род. в 1899, Северо-Казахстанская обл., Пресновский р-н, казах, обр.: среднее. Проживал: Северо-Казахстанская обл., Петропавловск. Третий секретарь, Петропавловский городской комитет ВКП(б).
Арестован УНКВД по СКО. 10.09.1937. Обв. по ст. 58-2, 58-7, 58-8, 58-11 УК РСФСР. Приговор: выездная сессия ВК ВС СССР, 09.03.1938 – ВМН.
Реабилитирован президиумом ВС КазССР 06.07.1957 за отсутствием состава преступления

Источник: Сведения ДКНБ РК по Северо-Казахстанской обл.
