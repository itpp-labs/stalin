МУРАВЬЕВ Даниил Иванович

Род. 12.12.1896, Тульская обл., Белявский р-н, дер. Марково, русский, обр.: низшее, член ВКП(б) с 1918 по 1938. Проживал: Винницкая обл., г. Уланов. Пред.Уланов.райисполкома Винниц.обл.
Арестован 03.06.1938. Обв. по ст. 54-2, 7, 8, 11 УК УССР. Приговор: ВК ВС СССР, 05.10.1938 – ВМН с конфискацией имущества.
Реабилитирован ВК ВС СССР 08.04.1958
Источник: УСБУ по Винницкой обл.
