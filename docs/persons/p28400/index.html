Дата сессии ВК ВС: 07.10.1938

Бисеров Мухамеджан (Мухаммеджон) Арифович
24.05.1898 г.р., м.р.: с. Авеево Елатомского уезда Тамбовской губ., из шахтерской семьи (выходец из крестьян?), образование: окончил 5 классов татарской сельской школы, в 1921-1922 гг. учился на рабочем факультете Ташкентского государственного университета, член ВКП(б)
с ноября 1929 г. до августа 1937 г. занимал руководящие должности, член Среднеазиатского бюро? в Ташкенте
Приговор: ВКВС СССР, 07.10.1938 — ВМН
Расстрелян 07.10.1938
Реабилитация: реабилитирован
