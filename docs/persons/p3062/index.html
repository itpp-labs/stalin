ЧЕЧЕЛЬ Василий Архипович

Род. в 1896, Украина, Днепропетровская обл., Перещепинский р-н, с. Голубовка, украинец, обр.: высшее, б/п. Проживал: Украинская ССР, г. Харьков, Володарского, 25, кв. 4. Экономист, нач. финанс. сектора обл. зам. управления
Арестован 11.02.1937. Обв. по ст. 54-7-10-11 (участник контрреволюционной организации правых, вел вредительскую работу, участвовал в подготовке террористических актов над руковод. сов. правительства"). Приговор: выездная сессия ВК ВС СССР, 08.07.1937 – ВМН. Расстрелян 08.12.1937, г.Харьков.
Реабилитирован 24.05.1958
Источник: База данных о жертвах репрессий Харьковской обл. (Украина)
