САМОЙЛОВ Давид Александрович

Род. 27.05.1880, г.Ейск Кубанской обл., еврей, из служащих, мещан, обр. высшее, Мос.техническое училище, инж.-теплотехник, б/п, в 1898-1906 член РСДРП(б), ст.инж. сектора эксплуатации Управления Мосэнерго, прож.: Москва, Покровка 41 - 47
Арестован 25.11.36 г. Обв. по ст.58-10 ч.1, подозрение в терор. деятельности, осужден по 58-9, 11. Приговор: ОСО при НКВД СССР, 23.07.1937 - 8 лет ИТЛ, направлен по этапу в Колымский край, бухта Нагаево. Умер 11.04.38 г. в Севвостлаге
Реабилитация: ВТ МВО, 30.08.56
Источник: Сведения М.И.Губиной, извлеченные из документов ГА РФ
