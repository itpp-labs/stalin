МАРКЕЛЫЧЕВ Дмитрий Михайлович

Род. в 1885, Ивановская обл., Пучевский уезд, Девкино, русский, обр.: высшее. Проживал: Восточно-Казахстанская обл., Зыряновское. Юрист Орса, Лесокомбинат.
Арестован УНКВД по Восточно-Казахстанской обл. 08.11.1936. Обв. по ст. 58-10 УК РСФСР. Приговор: ВК ВС СССР, 05.10.1937 – ВМН.
Реабилитирован Генеральной прокуратурой РК 28.05.1996 по Закону РК от 14.04.1993 г.
Источник: Сведения ДКНБ РК по г.Алматы
