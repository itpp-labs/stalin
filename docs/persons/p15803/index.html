ОБРЕЗАНОВ Василий Прохорович

Род. в 1902, Харьковская обл., Валковский р-н, г. Валки, украинец, член ВКП(б) с 1920 по 1937. Проживал: г. Харьков, ул. Чернышевского, 26 - 18. Нач.свеклотреста Харьковсвеклоснаба
Арестован УНКВД по Харьков.обл 24.10.1937. Обв. по ст. 54-7, 8, 11 УК УССР. 20.09.1939 – дело прекращено в силу ст.197 п.2 УПК УССР
Источник: УСБУ по Харьковской обл.
