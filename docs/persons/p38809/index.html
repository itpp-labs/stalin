Сулеев Билял Сулеевич
[1894 - .1938]
Род. в Капальском уезде Семиреченская обл., казах.
образование среднее,
зав. политпросветотделом Джетысуйского облоно (?-29.03.1923)
областной инспектор по просвещению Джетусуйской обл. (=04.1923=)
пред. обл. комиссией по устройству и реэвакуации беженцев (31.07.1923-
инспектор Наркомата просвещения по Джетысуйской обл - член облсовета союза бедноты "Кошчи" (=06.1924=),
инспектор Джетысуйского губоно (=12.1924-25=),
+ зав. Истпартом при Джетысуйском губкоме партии (12.1924-25 гг.),
член оргбюро казахского ташкентского землячества (06.1927-
+ член президиума Джетысуйского губ. комитета по латинизации каз. языка (04.1928-
зав. Алма-Атинским окроно (08.1928-10.1928 гг.),
зав. Актюбинским окроно (10.1928-
руководил передислокацией Казахского пединститута из Ташкента в Семипалатинск, директор Семипалатинского института
Арестован 27 марта 1931 г. в гор. Семипалатинск.
Приговорен "тройкой" при ПП ОГПУ в Казахстане 20 апреля 1932 г. по обв.: 58-11. В 1932 г. освобожден. Реабилитирован 28 февраля 1958 г. Верховный Суд КССР за отсутствием состава преступления.
