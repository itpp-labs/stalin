АЛИМОВ Камильджан

Список от 28 марта 1938г. [Узбекская ССР] - 1-я категория - РГАСПИ, ф.17, оп.171, дело 415, лист 222

7-й фонд ЦА ФСБ

Дата сессии ВК ВС: 08.10.1938

Дата расстрела: 08.10.1938

Алимов Камильджан

Род. в 1903 в Ташкенте в семье торговца-бакалейщика. Узбек. Чл. РКСМ с 1919. Чл. РКП(б) с января 1920. В 1913-1917 учился в старометодной и русско-туземной шк. в г. Мерв Закаспийской обл., с 1918 по 1919 в 35-й шк. для взрослых в Ташкенте, в 1922 окончил 1-й курс КУТВ им. Сталина, в 1925 1-й курс ГИЖ в Москве, в 1929 курсы м.-л. при ЦК ВКП(б) в Москве. С ноября 1917 по август 1918 уборщик гор. потреб. об-ва в Мерве. С октября 1918 по сентябрь 1921 учился и работал в Ташкенте. В октябре 1918 - марте 1919 учащийся 35-й шк. для взрослых. С марта по декабрь 1919 кучер-дворник 76-го квартирного комитета. С декабря 1919 по март 1920 секретарь горкома союза поваров. С марта 1920 по сентябрь 1921 организатор укома КСМ Туркестана. В сентябре 1921 - мае 1922 студент КУТВ им. Сталина в Москве. С мая 1922 по ноябрь 1924 работал в Ташкенте. В мае-сентябре 1922 инструктор райкома КСМ Туркестана. С сентября по ноябрь 1922 зав. политпросветотделом Сырдарьинского обкома КСМ Туркестана. С ноября 1922 по март 1923 ответредактор газеты "Ёш Шарк"1 ЦК КСМ Туркестана. С марта 1923 по июль 1924 секретарь ЦК КСМ Туркестана и ответредактор журнала "Узгаришчи Ёшлар"2. С июля по ноябрь 1924 зав. отделом печати ЦК КП(б) Туркестана. В ноябре 1924 - мае 1925 студент ГИЖ в Москве. По данным ОРПО ЦК ВКП(б) в 1924 исключался из ВКП(б). В мае 1925 - августе 1928 работал в Ташкенте. С мая 1925 по декабрь 1926 ответредактор редакции газет "Кизил Узбекистон", "Камбагал дехкон"3, журнала "Коммунист". С декабря 1926 по август 1928 зав. орграспредотделом и секретарь Ташкентского окружкома КП(б) Узбекистана. С сентября 1928 по июль 1929 слушатель курсов м.-л. в Москве. В июле 1929 - июле 1934 работал в Ташкенте. С июля 1929 по декабрь 1930 чл. правл. гос. треста "Средазводстрой". С декабря 1930 по январь 1933 упр. Среднеазиатским отд-нием Трактороцентра. С января 1933 по февраль 1934 уполн. НКЗ СССР по Средней Азии. С февраля по июль 1934 зам. зав. с.-х. отделом Средазбюро ЦК ВКП(б). С августа 1934 по декабрь 1936 2-й секретарь Кара-Калпакского обкома ВКП(б). В 1937 слушатель курсов переводчиков и редакторов в Москве. Арестован в Москве 1 октября 1937 ГУГБ НКВД УзбССР. Политбюро ЦК ВКП(б) 28 марта 1938 по списку санкционировало применение ВМН (1 кат.). Выездной сессией ВКВС СССР в Ташкенте 8 октября 1938 по обвинению в подготовке вооруженного восстания, террористической деятельности и участии в КРО по ст.ст. 58, 63, 64, 67 УК УзбССР приговорен к расстрелу, в тот же день расстрелян. Реабилитирован определением ВКВС СССР 2 марта 1957.

Источник: Филиппов С.Г. Территориальные руководители ВКП(б) в 1934-1939 гг.

Алимов Камильджан (Комилжон)

1903 г.р., м.р.: г. Ташкент, из рабочей семьи, образование: с 1910 г. один год учился в татарской мусульманской школе, затем 3 года - в Тухтамишской русско-туземной школе, с 1921 г. - в Коммунистическом университете трудящихся Востока в Москве, с 1924 г. -в Государственном институте журналистики в Москве., член ВКП(б) с 01.1920 г.
член совета национальностей ЦИК СССР
Приговор: ВКВС СССР, 08.10.1938 — ВМН
Расстрелян 08.10.1938
Реабилитация: реабилитирован
