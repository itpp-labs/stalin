ПЛЕТНЕР Исаак Маркович

Род. в 1897, Украина, Одесская обл., пос. Новоукраинка, еврей, из мещан, обр.: среднее, член ВКП(б) с 1924. Проживал: Украинская ССР, г. Харьков, Чернышевского, 65/67, кв. 5. Уполномоченный облзаготзерно
Арестован 08.08.1937. Обв. по ст. 54-8-10-11 (участник антисоветской правотроцкистской террористической организации). Приговор: спецколлегия Харьковского облсуда, 31.10.1937 – ВМН. Расстрелян 26.02.1938, г.Харьков.
Реабилитирован 29.08.1956
Источник: База данных о жертвах репрессий Харьковской обл. (Украина)
