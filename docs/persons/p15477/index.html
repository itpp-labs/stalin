Бондаренко Михаил Ильич

(26.8.1903, Елизаветград Херсонской губернии — 10.2.1938, Москва),
— партийный и государственный деятель.
Сын кузнеца.
С 1917 работал учеником молотобойца, токарем на заводе Эльворти.
В 1919—22 заведующий культотделом завкома.
В янв. 1925 вступил в РКП(б).
С 1925 заведующий отделом агитации и пропаганды комитета КП(б) Украины завода «Красная Звезда» (бывший Эльворти).
С 1926 заведующий отделом Исполкома Зиновьевского окружного совета.
С 1927 секретарь Компанеевского райкома КП(б) Украины.
С 1929 заведующий Отделом культуры Зиновьевского окружного Совета профсоюзов, в 1930 редактор окружной газеты.
С 1930 заведующий Культурно-пропагандистским отделом Зиновьевского горкома партии, с 1931 заместитель заведующего, заведующий Отделом агитации и массовых кампаний Закавказского крайкома партии.
С 1933 заведующий Орготделом Бакинского горкома, с 1934 1-й секретарь Азизбековского горкома, с 1935 — Шаумяновского райкома КП(б) Азербайджана.
С 4.12.1936 2-й секретарь Харьковского горкома, в июле—авг. 1937 — обкома КП(б) Украины.
В авг. 1937 1-й секретарь Винницкого обкома партии.
С 30.8.1937 член Политбюро ЦК КП(б) Украины и председатель СНК Украинской ССР.
13.10.1937 арестован.
8.2.1938 приговорен Военной коллегией Верховного суда СССР к смертной казни. Расстрелян.
14.4.1956 реабилитирован.

Бондаренко Михаил Ильич

Род. 26 августа (8 сентября) 1903 в г. Елизаветград Херсонской губ. в семье рабочего-кузнеца. Украинец. Чл. РКСМ с 1919. Канд. в чл. РКП(б) с января 1924, чл. РКП(б) с января 1925. В 1913 окончил нар. четырехклассную шк., в 1917 высш. нач. уч-ще, в 1922 кооперативный техникум в Елисаветграде. С 1915 по ноябрь 1926 работал и учился в Елизаветграде (Зиновьевске). В апреле-октябре 1915 ученик слесаря з-да с.-х. машиностроения акционерного об-ва " Р. и Т. Эльворти". С октября 1915 по май 1916 учащийся высш. нач. уч-ща. В мае-октябре 1916 ученик кузнеца з-да Эльворти". С октября 1916 по май 1917 учащийся высш. нач. уч-ща. С мая 1917 по май 1918 ученик кузнеца з-да Эльворти. В мае-сентябре 1918 рассыльный горкома Всероссийского союза рабочих металлистов, с сентября 1918 по октябрь 1919 зав. библиотекой Всероссийского Союза рабочих металлистов. С октября 1919 по сентябрь 1920 заготовитель отряда упродкома по продразверстке. В сентября 1920 - ноябре 1921 инспектор уездн. РКИ при СНХ. С ноября 1921 по май 1922 студент кооперативного техникума "Общее дело". С мая 1922 по июль 1923 политрук шк. ФЗУ, с июля 1923 по сентябрь 1924 зав. культотделом завкома з-да "Красная звезда" треста "Укрсельмаштрест". С сентября 1924 по февраль 1925 зав. окр. политпросветотделом. В феврале 1925 - январе 1926 зав. клубом Всесоюзного союза рабочих металлистов. С января по ноябрь 1926 секретарь партколлектива з-да "Красная звезда". С ноября 1926 по июль 1927 секретарь райкома КП(б)У в м. Верблюжка Зиновьевского окр. УССР, с июля 1927 по август 1928 секретарь Кампанеевского райкома КП(б)У в м. Кампанеевка Зиновьевского окр. С августа 1928 по январь 1931 работал в Зиновьевске. В августе 1928 - марте 1929 зав. культотделом окр. Совета профсоюзов. С марта 1929 по февраль 1930 редактор окр. газеты "Пролетарий". С февраля 1930 по январь 1931 зав. КПО горкома КП(б)У. С января 1931 по август 1933 работал в Тифлисе. С января по октябрь 1931 ответинструктор, с октября 1931 по сентябрь 1932 зам. зав. ОРГО, с сентября 1932 по август 1933 зав. АМО Закавказского крайкома ВКП(б). С августа 1933 по октябрь 1936 работал в Баку. С августа 1933 по февраль 1934 зав. ОРГО БК КП(б) Азербайджана. В феврале 1934 - июле 1935 1-й секретарь Азизбековского райкома КП(б) Азербайджана. С августа 1935 по октябрь 1936 1-й секретарь Шаумяновского райкома КП(б) Азербайджана. В октябре 1936 - августе 1937 работал в Харькове. В октябре-декабре 1936 зав. отделом парткадров, с декабря 1936 по июль 1937 2-й секретарь Харьковского горкома КП(б)У. С июля по август 1937 2-й секретарь Харьковского обкома КП(б)У. В августе 1937 1-й секретарь Винницкого обкома КП(б)У. С августа по октябрь 1937 пред. СНК УССР в Киеве. Арестован 13 октября 1937. Политбюро ЦК ВКП(б) 22 ноября 1937 по списку санкционировало применение ВМН (1 кат.). ВКВС СССР 8 февраля 1938 за активное участие в троцкистской террористической и вредительской АСО приговорен к расстрелу. Расстрелян 10 февраля 1938. Реабилитирован определением ВКВС СССР14 апреля 1956. Восстановлен в КПСС решением секретариата ЦК КПУ 6 июня 1956.

Источник: Филиппов С.Г. Территориальные руководители ВКП(б) в 1934-1939 гг.
