БЕКТЕНОВ Мендагали

Род. в 1905, Западно-Казахстанская обл., Джаныбекский р-н, аул Тани, казах, обр.: начальное. Проживал: Западно-Казахстанская обл. (Уральская), Джангалинский р-н, пос. Новая-Казанка. Заведующий
Арестован Джангалинский РО НКВД по ЗКО. 16.04.1938. Обв. по ст. 58-2, 58-7, 58-8, 58-11 УК РСФСР. Приговор: ВС СССР, 13.11.1938 – 12 лет ИТЛ.
Реабилитирован Верховным судом СССР 01.03.1958 за отсутствием состава преступления
Источник: Сведения ДКНБ РК по Западно-Казахстанской обл.
