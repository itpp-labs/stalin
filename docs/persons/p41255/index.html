Дамбинов Пётр Никифорович
родился 02(14).02.1892 в улусе Заглик Забайкальской области - расстрелян 14.10.1938, с 1917 по 04.1923 член Партии социалистов-революционеров
Образование
1907 - 1912\t учёба в Жердовском сельскохозяйственном училище (Иркутская губерния)
Послужной список
 - 1917\t учитель
1917\t товарищ председателя Иркутского отдела Бурят-Монгольского национально-революционного комитета
1919\t арестован, освобождён
3.6.1920 - 4.1921\t председатель Бурят-Монгольского национально-революционного комитета
\t товарищ председателя Учредительного Собрания Дальне-Восточной Республики
1920 - 1921\t редактор газеты «Голос бурят-монгола» (Чита)
.4 - 12.10.1921\t председатель Президиума Временного управления Бурят-Монгольской автономной областью Дальне-Восточной
\t   Республики
4.1921 - 21.10.1922\t товарищ председателя Народного Собрания Дальне-Восточной Республики
21.10.1922\t арестован в Дальне-Восточной Республике
\t выслан в РСФСР
1.1923 - 1.1925\t технический консультант по делам бурят-монголов при Постоянном представительстве Бурят-Монгольской АССР при
\t    Президиуме ВЦИК, старший хозяйственный агент Промбанка
