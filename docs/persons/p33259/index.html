ХОДОРОВ Семен Григорьевич

Список от 3 мая 1938г. [Московская область] - 1-я категория - РГАСПИ, ф.17, оп.171, дело 416, лист 338

Ходоров Семен Григорьевич
Род. в 1895, г.Кишинев (Бессарабия), еврей, из рабочих, обр. низшее, инженер, член ВКП(б) с 1918 г., исключен в сент.1937 г., чекист запаса, на момент ареста ареста диспетчер на 1 ГПЗ им.Кагановича, пом.нач. по механизации складского хоз-ва, ранее снят с работы Управляющего трестом "Союзстальмост" (до этого - дир. з-да "Янв.восст." в Одессе), прож.: Москва, 1-й Каптильский пер., д.9, кв.99, общежитие
Арестован 10.03.38 г. УНКВД МО, содержался в Таганской тюрьме. Обв. по ст.58-7, 11 19-58-8. Приговор: ВТ МВО, 31.01.1940 - по суду оправдать
Источник: Сведения М.И.Губиной, извлеченные из документов ГА РФ, дело № П-7825, т.1
