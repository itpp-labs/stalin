О проекте

ЛЯСКАЛО Георгий Федорович

Род. в 1887, Украина, г. Полтава, украинец, из служащих, член партии эсеров. Проживал: Украинская ССР, г. Харьков, Пушкинская, 49 а, кв. 40. Работал на трикотажной фабрике
Арестован 28.11.1937. Обв. по ст. 54-2-7-8-11 (принадлежность к антисоветской эсеровской организации). Приговор: тройка при УНКВД по Харьковской обл., 13.04.1938 – ВМН. Расстрелян 23.05.1938, г.Харьков.
Реабилитирован 29.01.1960
Источник: База данных о жертвах репрессий Харьковской обл. (Украина)
