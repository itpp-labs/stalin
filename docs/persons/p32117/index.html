УМАНСКИЙ Николай Семенович

Список от 3 мая 1938г. [Челябинская область] - 1-я категория - РГАСПИ, ф.17, оп.171, дело 416, лист 244

7-й фонд ЦА ФСБ

Дата сессии ВК ВС: 19.07.1938

Дата расстрела: 19.07.1938

Уманский Николай Семёнович

родился 27.7.1889 в Брацлаве Подольской губернии
1905 : член Польской социалистической партии
1907 : член Партии анархистов-максималистов
1919 : член РКП(б)
Послужной список
- 2.1918 : в русской армии
1918 - 1919 : председатель коллегии обвинителей Невельского уездного Революционного Трибунала (Витебская губерния)
1919 - 8.1920 : председатель Витебского губернского Революционного Трибунала
: заведующий Витебским губернским отделом юстиции
8 - 12.1920 : председатель Симбирского губернского Революционного Трибунала
12.1920 - 1921 : председатель Ставропольского губернского Революционного Трибунала
- 9.1921 : ответственный секретарь Ставропольского губернского комитета РКП(б)
10.1921 - 1923 : член Верховного Революционного Трибунала при ВЦИК
1.1923 - 5.1924 : член Кассационной коллегии по уголовным делам Верховного Суда РСФСР
5.1924 - : председатель Главного Суда Крымской АССР
- 9.1927 : в ВСНХ СССР
9.1927 - 3.1928 : болен, на излечении
3 - 11.1928 : заместитель заведующего Отделом судебного управления Народного комиссариата юстиции РСФСР
11.1928 - 8.1931 : председатель Дальне-Восточного краевого Суда
8.1931 - 5.1933 : председатель Нижне-Волжского краевого Суда
5.1933 - 3.1935 : председатель Карагандинского областного Суда
24.4.1935 - 10.1937 : прокурор Челябинской области
18.10.1937 : арестован
19.7.1938 : расстрелян
