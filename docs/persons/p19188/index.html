Сталинские расстрельные списки

РАСПЕВИН Николай Иванович

Род. в 1903, Нижегородская губ., Балахнинский уезд, дер. Сормово, русский, обр.: высшее, искл. из ВКП(б) 23 сентября 1937 г. за утерю партбилета. Проживал: Москва, ул. Большая Почтовая, д. 18/2, корп. 10, кв. 74. И.о. начальника Гл. инспекции спорта Всесоюзного комитета по делам физкультуры и спорта при СНК СССР.
Арестован 05.10.1937. Обв. в участии в антисоветской вредительской шпионской террористической организации. Приговор: ВК ВС СССР, 25.12.1937 – ВМН. Расстрелян 25.12.1937, г.Москва.
Реабилитирован ВК ВС СССР 31.03.1956
