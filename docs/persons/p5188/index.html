ШИ-ТО он же ВАН-ДЕ-ЦАЙ,

1892 г.рожд. уроженец Китая, кит. подданный, к моменту ареста без определенных занятий.

Р

Завербован для шпионской работы в пользу японцев в начале 1937 г. По заданиям ЛЮ-КО-ЖУНА, вел наблюдение за передвижением частей ОКДВА в погранполосе, неоднократно проникал на территорию военных аэродромов переда японцам сведения о типах самолетов и их численности.
