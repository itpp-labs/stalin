КАЛУГИН Михаил Степанович

Род. в 1910, д.1-я Красниковка Беседин.р-на Курской обл., русский, из рабочих, обр. 7 лет, член ВЛКСМ с 1931 г., шофер на легковой машине отдела охраны Яр.ж.д., прож.: Москва, В-Красносельская, 10 - 3а
Арестован 26.10.37 г. ДТО ГУГБ НКВД Яр.ж.д. Обв. по ст.17-58, 58-11 терроризм, диверсии. Приговор: выездная сессия ВК ВС СССР, 31.12.1937 - 10 лет, поражение в правах 5 лет
Источник: Сведения М.И.Губиной, извлеченные из документов ГА РФ
