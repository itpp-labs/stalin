Смолянский Яков Борисович

Род. в 1908, Херсон, еврей. Проживал: Москва.
Арестован в 1937. Обв. по ст. 58. Приговор: тройка при УНКВД по Дальстрою, 23.04.1938 – ВМН. Расстрелян 20.05.1938.
Реабилитирован 05.03.1962

Источник: Книга памяти Магаданской обл.

Смолянский Яков Борисович

Род. 24.11.1908, г.Херсон, еврей, из рабочих, обр. незаконченное высшее, отчислен в 1933 г. из Военно-химической академии (Москва) по болезни (без защиты диплома), в 1936 "учился экстерном в Менделеевском ин-те", исключен из ВКП(б) 22.02.1937, з-д №12, нач. школы ФЗУ, нач. отд. подготовки кадров. [В 1934 г. работал инженером в Военно-химическом тресте, где начальником был Сырцов. Затем Сырцов был директором з-да №12, Смолянский перешел на работу на з-д №12 и снимал комнату в квартире Сырцова.], прож.: Москва, ул.Кирова, 33-7
Арестован 10.02.37 г. Обв. по ст.58-10, КРД в военно-хим. промышленности. Приговор: ОСО при НКВД СССР, 14.07.1937 - 5 лет ИТЛ - Севвостлаг НКВД, бухта Нагаево
Обв.: а/сов.деятельность. Приговор: тройка УНКВД Дальстроя, 23.04.38 г. - ВМН

Источник: Сведения М.И.Губиной, извлеченные из документов ГА РФ
