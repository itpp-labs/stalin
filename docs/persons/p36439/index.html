САЛНА-ЗВЕДРИС Рудольф Янович

Род. в 1892, Лифляндская губ., Рижский уезд, усадьба Денди, латыш, обр.: среднее, член ВКП(б). Проживал: Москва, Смоленский бул., д. 5, кв. 2. Зам. секретаря Латвийской секции Коминтерна
Арестован 29.11.1937. Обв. в создании и руководстве к.-р. террористической организацией. Приговор: ВК ВС СССР, 07.09.1938 – ВМН. Расстрелян 07.09.1938, г.Москва.
Реабилитирован ВК ВС СССР 02.06.1956
