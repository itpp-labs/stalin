ИЗЮМСКИЙ Иван Петрович

из семьи зажиточных казаков. работал в городе на лесопильном заводе прорабом. Чл. ВКП(б) с 1918. Один из организаторов обороны Царицына, создатель царицынской Красной гвардии. Губернский комиссар, затем областной прокурор. В 1937 расстрелян, в 1954 реабилитирован.
